# 系统启动指南

## 快速启动（推荐）

### 一键启动

双击项目根目录下的 `start.bat` 即可自动启动前后端服务。

启动脚本会：
1. 自动清理占用的端口（8000、3000）
2. 启动后端服务
3. 启动前端服务
4. 显示服务地址

### 访问地址

- **前端页面**: http://localhost:3000
- **后端 API**: http://localhost:8000
- **API 文档**: http://localhost:8000/docs

---

## 手动启动（备用）

如果一键启动失败，可以手动启动：

### 后端

```cmd
cd /d E:\stock_predicition\backend
C:\Users\Lenovo\anaconda3\envs\finance\python.exe -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

**依赖**: 需要 conda 环境 `finance`

### 前端

```cmd
cd /d E:\stock_predicition\frontend
npm run dev
```

## 验证服务状态

启动后，访问以下接口验证：

```bash
# 查看系统信息
curl http://localhost:8000/

# 查看数据状态
curl http://localhost:8000/api/v1/admin/data-status
```

预期返回：
```json
{"code":200,"message":"success","data":{"stocks":10,"comments":xxx,"sentiments":xxx,"has_data":true}}
```

## 常见问题

### Q: 一键启动失败怎么办？

A: 使用手动启动方式，确保 conda 环境 `finance` 已激活。

### Q: 端口被占用？

A: `start.bat` 会自动清理旧进程，也可以手动结束：
```cmd
taskkill /F /IM python.exe
taskkill /F /IM node.exe
```

### Q: conda 环境不存在？

A: 创建 conda 环境：
```cmd
conda create -n finance python=3.10
conda activate finance
pip install -r requirements.txt
```
