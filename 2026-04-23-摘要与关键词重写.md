# 股票情绪预测系统论文重写

## 摘要

中国A股市场投资者人数众多，社交平台上每天产生的海量评论中蕴含着丰富的市场情绪信息。然而，传统量化策略受限于价量数据，对情绪脉冲的捕捉往往力不从心。本文基于Transformer架构设计并实现了一套面向A股投资者的多模型情绪预测系统，采用哈工大讯飞实验室发布的Chinese-BERT-wwm-ext预训练模型，结合FinBERT及金融情感词典，构建三模型协同的文本分类框架。在此基础上，本文建立了包含看涨指数、看跌指数、情绪强度、情绪温度和情绪波动五个维度的情绪指标体系，并通过Pearson相关分析和格兰杰因果检验验证情绪信号与股价走势的统计关联。系统后端采用FastAPI构建RESTful API，前端基于Vue3实现可视化展示，底层数据持久化依托MySQL实现，缓存加速由Redis负责，异步任务调度通过Celery完成，构建了从数据采集、文本清洗、情感分析、指标量化到结果展示的完整技术闭环。研究表明，基于多模型融合的情绪分析能够有效捕捉市场情绪变化，为量化投资策略提供新的信号来源。

## 关键词

股票情绪分析；Transformer；多模型对比；Chinese-BERT-wwm-ext；格兰杰因果检验

## Abstract

The A-share market in China has a huge investor base, generating massive comment data on social platforms daily that contain rich market sentiment information. Traditional quantitative strategies, constrained by price-volume data, often fail to capture sentiment pulses effectively. This paper designs and implements a multi-model stock sentiment prediction system based on the Transformer architecture for A-share investors. The system employs Chinese-BERT-wwm-ext from HIT-SCIR, along with FinBERT and a financial sentiment lexicon, forming a three-model collaborative text classification framework. Based on this, we construct a five-dimensional sentiment indicator system comprising bull index, bear index, sentiment intensity, sentiment temperature, and sentiment volatility. Pearson correlation analysis and Granger causality test are applied to examine the statistical relationship between sentiment signals and stock price movements. The backend is built with FastAPI, the frontend visualizes data using Vue3, MySQL handles persistent storage, Redis provides caching, and Celery manages asynchronous task scheduling. This forms a complete technical chain from data collection, text preprocessing, sentiment analysis, indicator quantification to result presentation. Our research demonstrates that multi-model fusion-based sentiment analysis can effectively capture market sentiment changes, providing novel signal sources for quantitative investment strategies.

**Keywords**: Stock Sentiment Analysis; Transformer; Multi-Model Comparison; Chinese-BERT-wwm-ext; Granger Causality Test
