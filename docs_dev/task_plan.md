## Phase 1: 丰富预警规则

**文件**: `backend/app/services/alert/engine.py`

新增规则：
1. `sentiment_reversal` - 情绪趋势反转（连续N天同向后突然反向）
2. `burst_activity` - 舆情爆发（短时间内评论激增）
3. `sector_linkage` - 多股票联动（同行业同时异常）
4. `volatility_anomaly` - 波动率异常
5. `new_high_low` - 极值突破（新低/新高检测）
